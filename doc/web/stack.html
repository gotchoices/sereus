Stack

The Sereus stack composes well-known building blocks and original components:

  • Quereus: SQL query processor.
  • Optimystic: B-trees, indexes, block storage.
  • Fret: Distributed hash table (part of Optimystic).
  • libp2p: Peer-to-peer transport.
  • Bootstrap: Initialization and developer tooling.
