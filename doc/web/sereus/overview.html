Overview

Sereus is a Web3 fabric where identities and permissions are scoped to consent-based threads. Each thread is a shared SQL database among its participants with role-based access.

  • No global identity leakage; only thread participants see each other.
  • Invite/consent flows form trusted subnets, not public broadcast.
  • Data lives in shared SQL, replicated across participants' cadres.
